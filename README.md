## Whistleblower

WhistleblowerEscrow is an on-chain escrow + access-control contract
that lets whistleblowers submit encrypted information (via IPFS),
and lets a trusted agent sell access to journalists in a controlled way.

## Roles

Whistleblower → submits info
Agent → approves access & handles payment
Journalist → gets access off-chain
Owner → governance (updates agent)
